## 🇺🇸 English

**Overy** is a powerful macOS utility that translates text instantly just by hovering your mouse cursor over it. No need to copy-paste or switch apps. It integrates seamlessly with your workflow, supporting multiple translation engines like DeepL, Google Translate, and MyMemory.

### ✨ Key Features
- **Instant Hover Translation**: Simply hover your mouse over any text in any app to see a translation popup.
- **Multiple Engines**: Support for **DeepL** (High quality), **Google Translate**, and **MyMemory** (Free, no key required).
- **Global Shortcut**: Toggle translation mode with `CMD + Shift + T`.
- **Translation History**: Automatically saves your translation history for later review.
- **Auto-Copy**: Option to automatically copy translated text to the clipboard.
- **Customizable**: Adjust hover delay, shortcut keys, and UI languages.

### 🚀 Getting Started
1. **Download** the latest release from the Releases page.
2. **Move to Applications** folder and launch Overy.
3. **Grant Accessibility Permission**: Overy needs permission to read text under your cursor. Follow the prompts to enable it in System Settings.
4. **Activate**: Press `CMD + Shift + T` or click the menu bar icon to start translating!

### 🛠️ Configuration
- **API Keys**: For DeepL and Google Translate, you can enter your own API keys in Settings > Translation Service.
- **Languages**: Supports source language auto-detection and customizable target languages.

## 🇰🇷 한국어

**Overy**는 마우스 커서를 텍스트 위에 올리기만 하면 즉시 번역해주는 강력한 macOS 유틸리티입니다. 복사-붙여넣기나 앱 전환 없이, 작업 흐름을 방해하지 않고 매끄럽게 번역하세요. DeepL, Google 번역, MyMemory 등 다양한 번역 엔진을 지원합니다.

### ✨ 주요 기능
- **즉시 마우스 오버 번역**: 어떤 앱에서든 텍스트 위에 마우스를 올리면 번역 팝업이 나타납니다.
- **다양한 엔진 지원**: **DeepL** (고품질), **Google 번역**, **MyMemory** (무료, 키 불필요)를 지원합니다.
- **글로벌 단축키**: `CMD + Shift + T`로 언제든지 번역 모드를 켜고 끌 수 있습니다.
- **번역 히스토리**: 번역된 내용은 자동으로 기록되어 나중에 다시 볼 수 있습니다.
- **자동 복사**: 번역 결과를 클립보드에 자동으로 복사하는 옵션을 제공합니다.
- **사용자 맞춤 설정**: 마우스 오버 대기 시간, 단축키, UI 언어를 자유롭게 설정하세요.

### 🚀 시작하기
1. Releases 페이지에서 최신 버전을 **다운로드**합니다.
2. **응용 프로그램** 폴더로 이동 후 실행합니다.
3. **손쉬운 사용 권한 허용**: 커서 아래의 텍스트를 읽기 위해 권한이 필요합니다. 안내에 따라 시스템 설정에서 허용해주세요.
4. **활성화**: `CMD + Shift + T`를 누르거나 메뉴 막대 아이콘을 클릭하여 번역을 시작하세요!

### 🛠️ 설정
- **API 키**: DeepL 및 Google 번역 사용 시, 설정 > 번역 서비스에서 자신의 API 키를 입력할 수 있습니다.
- **언어**: 원본 언어 자동 감지 및 원하는 도착 언어 설정을 지원합니다.

## 🇯🇵 日本語

**Overy**は、マウスカーソルをテキストの上に置くだけで瞬時に翻訳する強力なmacOSユーティリティです。コピー＆ペーストやアプリの切り替えは不要。DeepL、Google翻訳、MyMemoryなど、複数の翻訳エンジンをサポートし、作業フローにシームレスに統合されます。

### ✨ 主な機能
- **インスタントホバー翻訳**: どのアプリでもテキストの上にマウスを置くだけで翻訳ポップアップが表示されます。
- **マルチエンジン対応**: **DeepL**（高品質）、**Google翻訳**、**MyMemory**（無料、キー不要）をサポート。
- **グローバルショートカット**: `CMD + Shift + T`で翻訳モードのオン/オフを切り替えられます。
- **翻訳履歴**: 翻訳履歴は自動的に保存され、後で確認できます。
- **自動コピー**: 翻訳結果をクリップボードに自動的にコピーするオプションがあります。
- **カスタマイズ可能**: ホバー待機時間、ショートカットキー、UI言語を調整できます。

### 🚀 始め方
1. Releasesページから最新バージョンを**ダウンロード**します。
2. **アプリケーション**フォルダに移動して起動します。
3. **アクセシビリティ権限の許可**: カーソル下のテキストを読み取るために権限が必要です。システムの指示に従って有効にしてください。
4. **有効化**: `CMD + Shift + T`を押すか、メニューバーのアイコンをクリックして翻訳を開始しましょう！

### 🛠️ 設定
- **APIキー**: DeepLおよびGoogle翻訳を使用する場合、設定 > 翻訳サービスで独自のAPIキーを入力できます。
- **言語**: ソース言語の自動検出と、カスタマイズ可能なターゲット言語をサポートしています。

## 🇨🇳 中文

**Overy** 是一款强大的 macOS 实用工具，只需将鼠标光标悬停在文本上即可即时翻译。无需复制粘贴或切换应用。它无缝集成到您的工作流程中，支持 DeepL、Google 翻译和 MyMemory 等多种翻译引擎。

### ✨ 主要功能
- **即时悬停翻译**: 在任何应用中，只需将鼠标悬停在文本上即可看到翻译弹窗。
- **多引擎支持**: 支持 **DeepL**（高质量）、**Google 翻译** 和 **MyMemory**（免费，无需密钥）。
- **全局快捷键**: 使用 `CMD + Shift + T` 随时切换翻译模式。
- **翻译历史**: 自动保存翻译历史，方便稍后查看。
- **自动复制**: 提供自动将翻译结果复制到剪贴板的选项。
- **可自定义**: 调整悬停延迟、快捷键和界面语言。

### 🚀 快速开始
1. 从 Releases 页面**下载**最新版本。
2. 移动到**应用程序**文件夹并启动。
3. **授予辅助功能权限**: Overy 需要权限来读取光标下的文本。请按照提示在系统设置中启用。
4. **激活**: 按 `CMD + Shift + T` 或点击菜单栏图标开始翻译！

### 🛠️ 配置
- **API 密钥**: 对于 DeepL 和 Google 翻译，您可以在设置 > 翻译服务中输入自己的 API 密钥。
- **语言**: 支持源语言自动检测和自定义目标语言。
